# Gradient_Descent_Momentum

## 경사하강법 모멘텀

1. 각 반복에서 갱신된 최적화 변수 'theta'값 저장
2. 모멘텀 알고리즘은 기울기 값의 지수 가중 평균을 이용하여 현재 모멘텀 벡처를 계산
3. 'v'는 이전 모멘텀 벡터에 현재 기울기 값을 가중 평균한 값
4. 'x'를 업데이트할 때, 이전 모멘텀 벡터를 더해주어 모멘텀 적용
